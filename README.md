# Support Matrix Frontend

RISC-V Board and OS Support Matrix Website

## Develop

```bash
git clone git@github.com:QA-Team-lo/support-matrix-frontend.git 

cd ./support-matrix-frontend

git submodule update --init

pnpm install

pnpm run dev
```
